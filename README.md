# CMR TTT
## Zum Server verbinden
- Garry's mod starten
- Im Spiel auf `Find Multiplayer Game` clicken
- Auf `Trouble in Terrorist Town` clicken
- `Pebkac` suchen
- Passwort aus der Beschreibung vom Garry's Mod Channel auf dem TS eingeben
